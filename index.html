  • Filtering

    System tags

    Set the system tags you want to allow when requesting images. Click to toggle.


    Custom filter

    By default search results use the "Everything" filter, which means that you may see images having tags which you've hidden.

    To use a specific filter for fetching images, turn on the option below, then select from your filters or enter its ID.

    Keep in mind that you still need to enable system tags you wish to see regardless of your filter setting.

    The image will be updated in to reflect the changes.
  • Metadata

    Choose what information you want to see in the bottom right corner.

  • Image cropping

    This setting allows you to modify how the image is displayed if its size differs from the browser's size (which happens most of the time).
